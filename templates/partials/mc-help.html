Character Sheet Help

This page will explain how to use this character sheet.

Editing

Throughout the character sheet you will see the icon. Clicking this will allow you to edit that section. Once in edit mode the icon will turn solid (). Many areas are purposefuly made so that it requires to be in edit mode to change, so that mistake isn't accidentally made when viewing regulary.

You can have more than one section/item in edit view at the same time.

Shutdown

Throughout the character sheet you will see the icon. This means that the section or item is "on". Clicking this will allow you to shutodwn that section or item. Once shutdown the icon will change to the off position (). While shutdown, any rollable traits will not add to the Dice Pool. Also, section or item will be faded while in read view.

Detail Lists

A few items thorughout the character sheet may have the options of "Detail Lists". Whether or not these are present are determined in the System Settings. Things like traits and signature assets may have this option.

The point of detail lists is to be a catch all for things like tags, SFX, and any other lists you may add to an item.

Detail Lists can have any amount of list items.

Name

This is intended to be the unique name of the detail item, it will appear in bold

Type

This is intended to be the category or tag type to the detail item, e.g. SFX, limit, etc. It will appear in bold and italic.

Unlock

When checked, the icon before the detail item in read view will be a solid circle: . When unchecked it will be an unfilled circle:

This option's intended purpose is to show whether or not an SFX is unlocked. However, you could of course use it for a different purpose, depending on your context.

Value

This can be a description, a comma separated list of values, etc. This obviously depends on the context and what the detail's item type is.

Plot Point

The Plot Points section of the character sheet is fairly simple to use. Simply click on the big PP symbol to spend a plot point, assuming you have one. The number below the icon indicates how many plot points you have. Clicking on the icon will add a plot point.

Adding or spending a plot point will display the appropriate message in the chat.

Plot Point spending is hooked up with the DiceSoNice module to show a PP token when spent.

Shutdown (/)

Clicking this will allow you to shutdown or reverse the shutdown of this trait set and all of its associated traits. For more detailed information, see the shutdown section above.

Add Trait (green plus button)

Clicking this icon will allow you to add a trait to the trait set. Whether or not you can add a trait to a trait set depends on the settings in the System Settings set up by the GM. If you cannot add a trait, this button will not appear.

Set Description

A general description of the trait set.

Traits

Each trait may have different options for the associated trait set depending upon the System Settings. However, below is a description of each option you may find in a trait and it's subsequent traits.

Trait Name

The name of your trait.

Die Rating

In read view it simply shows what die or dice you have associated with this trait. You may change the die rating or add dice to the trait in the edit view. What die ratings you may choose, minimum and maximum dice you may have on a trait in this trait set is determined by the trait set settings in the System Settings, set up by the GM.

Shutdown (/)

Clicking this will allow you to shutdown or reverse the shutdown of this trait. For more detailed information, see the shutdown section above.

Trait Description

A general description of the trait.

Detail List

Some trait sets allow detail lists in their traits as determined in the System Settings by the GM. For more information on Detail Lists see Detail Lists above.

Misc Traits

The miscellaneous traits appear on the right side of the character sheet (e.g. Signature Assets, Assets, Complications, etc.). Some of these miscellaneous traits will not appear if they are not included by the GM in the System Settings.

The options that you see in these traits work essentially the same as those in the Traits section above.

Dice Pool

This area of the character sheet is to prepare and modify your dice for rolling.

Add Custom Dice

Here you can add any die or dice to your pool. You can optionally use the label field to add a label as it appears in your pool. You can also modify or add dice to this label. There is a reset button which will clear out the label for you and set the die configuration to a d8 for your next add. Finally, there is an add button to add your custom die/dice to the dice pool.

Gray Pool Box

This is where all dice you have selected from your various traits (by clicking on their names while in the read view), as well as any custom dice you may have added. While in this box you can modify the dice or add more to them.

At the bottom right is a clear button to clear the pool of any dice and a roll button to roll the dice in the dice pool.
